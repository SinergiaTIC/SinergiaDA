Sobre EDA

EDA (Enterprise Data Analytics) es un software de código abierto utilizado principalmente para la muestra de datos empresariales de forma gráfica y visual. Se basa en un sistema de informes, los datos de cada informe se distribuyen en paneles y estos tienen la posibilidad de modificarse para mostrar de distintas formas dicha información.

También tendremos la posibilidad de añadir o modificar fuentes de datos. En EDA hay muchas utilidades para configurar y personalizar cómo deben mostrarse los campos, así que en este aplicativo tendremos la oportunidad de asignar permisos, extensiones de modelo, restricciones entre otras muchas utilidades diversas para las entidades o tablas que necesitemos transformar.

Por último cabe decir que EDA ofrece un sistema de gestión de usuarios, en este existen las funciones usuales de crear,actualizar,eliminar o visualizar los permisos de un usuario, un apartado donde crear grupos para administrar categorías de usuarios con permisos concretos, un pequeño gestor de correos y por último un apartado donde poder exportar modelos y informes en relación a los datos guardados.
