Para usar OAuth2 con Gmail:
  1. Ve a Google Cloud Console
  2. Crea o selecciona un OAuth Client ID
  3. Añade http://localhost a los "URI de redireccionamiento autorizados"
  4. Habilita la API de Gmail
  5. Copia aquí tu Client ID y Client Secret
El Refresh Token se obtiene haciendo clic en "Obtener Refresh Token".
