Eda can be translated by running the TransalteEda.jar tool
It requires java 8

It will scan for translation files located in this folder. they must be called eda_translation_XX.txt where XX is the locale for the app translation and api_translation_XX.txt for the api translation.

How to translate it.

1. run the Translator:
java -jar TranslateEda.jar
2. Follow the instructions
3. Compile de app 
ng build --prod
4. Run the app or build the docker.
